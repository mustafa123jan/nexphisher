* `apt update`
* `apt install git -y`
* `git clone git://github.com/mustafa123jan/nexphisher.git`
* `cd nexphisher`
#### > SETUP : `bash setup`
#### > SETUP [TERMUX] : `bash tmux_setup`
#### > Run : `bash nexphisher`
